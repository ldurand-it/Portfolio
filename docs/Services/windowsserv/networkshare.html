Partage Réseau sur Windows Server 2022

Introduction & Prérequis​

La mise en place d'un partage réseau sous Windows Server 2022 est une étape cruciale pour permettre le partage et l'accès efficace des fichiers au sein d'un réseau local. Ce guide détaille les étapes nécessaires pour configurer un partage réseau via le Gestionnaire de Serveur, offrant une solution rapide et sécurisée pour partager des ressources au sein de votre environnement Windows.

Prérequis​

Avant de commencer le processus de configuration du partage réseau, assurez-vous d'avoir les éléments suivants en place :

  • Windows Server 2022 installé et opérationnel :

Votre serveur Windows doit être installé et fonctionner correctement.

  • Accès au Gestionnaire de Serveur :

Assurez-vous d'avoir un accès administratif au Gestionnaire de Serveur.

  • Services de Fichiers et de Stockage :

Les Services de Fichiers et de Stockage doivent être activés sur votre serveur.

  • Comptes Utilisateurs :

Préparez les comptes d'utilisateurs qui auront accès au partage, le cas échéant.

  • Volumes Disponibles :

Au moins un volume doit être disponible sur lequel vous souhaitez créer le partage.

Assurez-vous de disposer de ces éléments avant de suivre les étapes du guide pour garantir une configuration sans accroc de votre partage réseau. 🚀📂

Gestionnaire de serveur​

Dans un premier temps lancer l'application "Gestionnaire de Serveur" sur votre Serveur.

Services de Fichiers & Stockage​

Dans le Gestionnaire de Serveur, sélectionnez "Services de fichiers et de stockage".

Créer nouveau Partage​

Sous "Partages", effectuez un clic droit, puis choisissez "Nouveau Partage".

Configuration Rapide de SMB​

Choisissez "SMB Rapide" pour une configuration rapide.

Sélectionner le Volume​

Sélectionnez le volume que vous souhaitez partager.

Nommer le Partage​

Nommez votre partage de manière descriptive.

Autoriser la Mise en Cache du Partage​

Activez l'option pour autoriser la mise en cache du partage.

Configurer les Autorisations​

  • Sous "Autorisations", cliquez sur "Personnaliser", puis désactivez l'héritage du compte Admin.

  • Supprimez les objets hérités et ajoutez les comptes d'utilisateurs nécessaires.

Finaliser la Configuration​

Cliquez sur "Suivant", vérifiez les configurations, puis cliquez sur "Créer" pour finaliser le partage.

Résultat​

Bien Joué ! ✔️

Félicitations ! Vous avez maintenant configuré avec succès un partage réseau sur votre Windows Server 2022. Assurez-vous de tester l'accès au partage depuis d'autres ordinateurs pour garantir son bon fonctionnement. 🌐🚀
