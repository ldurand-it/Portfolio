Joindre une machine Windows à un Domaine

Introduction & Prérequis​

Bienvenue dans ce guide détaillé qui vous accompagnera à travers le processus de jonction d'une machine Windows à un domaine.
L'intégration à un domaine permet une gestion centralisée des utilisateurs, des groupes et des politiques, facilitant ainsi l'administration au sein de votre environnement réseau.

Prérequis​

Ce guide s'intègre dans la suite de guides sur Windows Server 2022 qui sont respectivements les suivants : Création d'une Machine Virtuelle Windows Server 2022 (ProxmoxVE), Active Directory Windows Server 2022 & DHCP sur Windows Server 2022
Avant de commencer, assurez-vous d'avoir les informations suivantes :

  • Nom du domaine auquel vous souhaitez joindre la machine.
  • Identifiants d'administrateur de domaine.

Joindre le Domaine​

  • Windows + R
  • Tapez sysdm.cpl
  • Une fenêtre propriétes système s'ouvre
  • Cliquez sur Modifier
  • Selectionner Domain puis renseingner votre domaine
  • Validez avec "OK"
  • Renseigne ensuite le compte Administrateur du Domaine
  • Patientez un court insant et vous aurrez un message de Bienvenue dans votre nouveau domaine

Résultat​

  • Après le redémarrage, connectez-vous en utilisant un compte du domaine.

Conclusion​

tip

Félicitations ! Vous avez maintenant réussi à joindre votre machine Windows à un domaine. Profitez des avantages d'une gestion centralisée et simplifiée au sein de votre réseau. N'hésitez pas à explorer les fonctionnalités du domaine pour tirer le meilleur parti de cette intégration. 🌐🤝
