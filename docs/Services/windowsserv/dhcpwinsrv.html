DHCP sur Windows Server 2022

Introducion & Prérequis​

Bienvenue dans ce guide pratique qui vous aidera à mettre en place un serveur DHCP (Dynamic Host Configuration Protocol) sur votre réseau. Suivez ces étapes simples pour garantir une distribution dynamique et efficace des adresses IP à vos clients. Suivez attentivement les instructions ci-dessous pour établir un serveur DHCP opérationnel et assurez-vous de profiter pleinement des avantages d'une allocation dynamique des adresses IP. 🌐🚀

C'est quoi le DHCP ?

La fonction Dynamic Host Configuration Protocol (DHCP) est un protocole client/serveur qui fournit automatiquement une adresse Internet Protocol (IP) et d'autres informations de configuration pertinentes à un hôte IP (par exemple, masque de sous-réseau et passerelle par défaut).

Prérequis​

Dasn ce guide nous faisons suite à ces deux guides précédent : Création d'une Machine Virtuelle Windows Server 2022 (ProxmoxVE) & Active Directory Windows Server 2022.
Pour configurer votre serveur DHCP sur Windows Server 2022 vous aurez donc besoin d'un domaine déjà configuré et d'une machine Windows 10/11 pour tester le DHCP.

Installation de la fonctionnalitée DHCP​

  • Ouvrez le "Gestionnaire de Serveur".
  • Sélectionnez "Ajouter des Rôles et Fonctionnalités".
  • Dans "Rôle de Serveur" sélectionnez "DHCP".
  • Suivez l'assistant pour terminer l'installation.

Configuration du DHCP​

Après l'installation ne fermez pas la fenêtre et cliquez sur Terminer la configuration DHCP. On va devoir autoriser notre serveur DHCP sur notre domaine pour cela gardez la première Option qui est configurée avec le compte Adminsitrateur du domaine et faites valider.
Là après un très court chargement dans la fenêtre "Résumé" vous verrez deux étapes qui sont en mode terminés .

Création d'une nouvelle étendue​

En créant une nouvelle étendue, donnez-lui un nom significatif et décrivez son objectif. Définissez la plage d'adresses à distribuer et ajustez la durée du bail

  • Suivez l'assistant de création de nouvelle étendue.
  • Entrez un nom et une description pour l'étendue.
  • Définissez la plage DHCP (par exemple, de 192.168.50.10 à 192.168.50.20).
  • Aucune exclusion nécessaire.
  • Définissez la durée du bail à 4 heures.
  • Mettre une Passerelle par défaut (par exemple 192.168.50.254)

Les Options DNS & WINS​

DNS​

On laisse que qui est configuré par défaut tout en vérifiant que ce sont biens les infos que l'on veut transmettre au client de notre DHCP

WINS​

On ne touche pas à cette options ce n'est pas nécéssaire pour un fonctionnement simple que nous mettons en place dans ce guide.

Activer l'étendue​

Pour activer votre étendue rien de plus simple que de suivre l'assistant qui à la fin vous proposera d'activer l'étendue

Résultat​

Phase de Test​

  • Sur une machine cliente Windows, ouvrez une invite de commande.
  • Tapez ipconfig /all pour afficher la configuration actuelle.
  • Tapez ensuite ipconfig /renew pour renouveler l'adresse IP.

Conclusion​

Bien Joué ! ✔️

Félicitations ! Vous avez maintenant un serveur DHCP opérationnel sur votre réseau. Les clients qui rejoignent votre réseau recevront automatiquement des adresses IP conformes à la configuration que vous avez définie. N'hésitez pas à ajuster les paramètres selon les besoins spécifiques de votre environnement. 🚀
