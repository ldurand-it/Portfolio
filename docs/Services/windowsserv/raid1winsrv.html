Disques durs en RAID 1 sur Windows Server 2022 (PVE)

Introduction & Prérequis​

Bienvenue dans ce guide qui vous guidera à travers la mise en place d'un RAID 1 sur un Windows Server tournant sous Proxmox.
Suivez attentivement ces étapes pour assurer une configuration réussie de votre RAID.

RAID 1

La configuration RAID 1 permet de sécuriser un système en disposant de deux disques avec exactement les mêmes données. Dans cette configuration on ne recherche pas la performance mais plutôt la sécurité. Soit une donnée A et une donnée B.

Prérequis​

Avant de débuter le processus de configuration du RAID 1 sur votre machine virtuelle Windows Server tournant sous Proxmox, assurez-vous d'avoir satisfait aux prérequis suivants :

  • Une Machine Virtuelle Windows Server 2022
  • Possibilité de lui ajouter 2 Disques avec suffisemment d'espace

Ajouter Deux Disques à la VM​

  • Ajoutez deux disques de 5 Go en tant que périphériques IDE à votre machine virtuelle Proxmox.
Taille

Ajustez la taille de vos disques en fonction de vos besoins.

warning

La taille de vos deux disques doit être similaires

Appliquer les modifications​

  • Arrêtez et redémarrez la machine virtuelle pour appliquer les modifications.
Modifications

Lorsque vous modifiez la configuration de votre VM elles apparaissent en Orange quand elle ne sont pas encore prisent en comptes.

Configuration dans Windows Server​

  • Cliquez avec le bouton droit de votre souris sur l'icône Windows.
  • Sélectionnez "Gestion des disques" dans le menu

Initialisation des disques​

  • Initialisez les disques en utilisant le format de partition GPT.

Création d'un volume en Miroir (RAID 1)​

  • Cliquez avec le bouton droit de votre souris sur l'un des deux disques initialisés.
  • Choisissez "Nouveau Volume en Miroir".
  • Déplacez les deux disques dans la catégorie "Sélectionnés".

Attribuer une lettre de lecteur​

  • Attribuez une lettre de lecteur au volume miroir.

Formatage et Nom du Volume​

  • Formattez le volume en NTFS et donnez-lui un nom, par exemple, "DATA".

Valider l'oppération​

  • Vérifiez les configurations et terminez le processus.

Résultat​

Après le formattage, un disque "DATA" de 5 Go devrait apparaître dans Ce PC.

Bien Joué ! ✔️

Félicitations ! Vous avez maintenant mis en place un RAID 1 sur votre Windows Server tournant sous Proxmox. Assurez-vous de tester la redondance et la fiabilité du RAID pour garantir son bon fonctionnement. 🚀🔧
