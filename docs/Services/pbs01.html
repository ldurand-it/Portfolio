Sauvegarde des VM sur PVE avec Proxmox Backup Server (PBS)

Introduction & Prérequis​

Avec la numérisation grandissante du monde qui nous entoure et la multiplication des catastrophes naturelles mais aussi des cyberattaques il est désormais primordiale pour une entreprise de posséder des sauvegardes de son activités. Pour cela il faut mettre en place un système de sauvegarde efficace sans erreur et régulier. Il faut aussi tenter le plus sovent possible de retaurer ses Backups.

Travaillant presque exclusivement avec la solution Proxmox Virtual Envrionnement pour déployer des machines virtuelles. J'ai désormais besoin de sauvegarder ces VM's pour pouvoir les restaurer rapidement à l'état dans lesquels je les avais précedemment perdu.

Pour cela j'utilise Proxmox Backup Server qui est la solution faite pour Proxmox Virtual Environnement.

Prérequis​

Dans ce guide je vais utiliser :

  • 1 PVE avec dessus une VM à sauvegarder
  • 1 PBS sur le même réseau que mon PVE
  • 3 Disks virtuelles sur le PBS avec 1 de 8GO pour l'OS et deux autres de 20GO chacun pour créer un miroir et sauvegarder dessus.

Installation de PBS​

Alors commençons par installer la solution Proxmxox Backup Server. J'ai donc créer une Machine Virtuelle. Dans l'idéal évidemment utiliser un environnemment différent de celui de production. Là pour le test j'installerais ça sur une VM Proxmox. On configure donc la VM d'une façon particulière pour les Disks.

Voici à quoi ça ressemble :

Quand la machine est prête vous pouvez alors la lancez et débuter l'installation de Proxmox Backup Server.

  • Séléctionnez l'option Installation Graphique

  • Acceptez le contrat de license

  • Séléctionnez le disque de 8 GO comme sible au format ext4 pour installer le système

  • Renseignez le mot de passe Root et un email

  • vérifier les informations renseignés et confirmez les

  • L'installation se lance et se termine

Interface Web de PBS​

Pour configurer Proxmox Backup Server on utilise comme PVE son interface Web. Pour avoir accès à cette interface vous devez vous rendre à l'IP de votre PBS et utiliser le Port 8007

Utiliser les identifiants root avec le mot de passe que vous avez renseigné.

Voici l'interface Web après vous être connecter.

Création du Datastore​

Pour créer notre Datastore on se rend dans la catégorie Administration > Storage/Disks > ZFS

Cliquez sur Create ZFS.

Effectuez la même config.

Connexion PBS vers PVE​

Dans cette étape nous allons connecter notre système de sauvegarde PBS à notre Hyperviseur PVE. Pour cela il va falloir récupérer le fingerprint du PBS afin de le raneseigner dans un endroit bien spécifique sur PVE. Sur la section Dashboard cliquez sur Show Fingerprint et copiez le.

Ensuite sur PVE dans Datacenter > Storage > Add Proxmox Backup Server

Remplissez avec vos infos et déposer votre Fingerprint

Pour la Backup Retention cela se gère dans PBS directement, ici laisser l'option Keep All Backups activer

Configurer les Tâches de Sauvegardes​

Prune Job​

Le Prune Job se configure sur le Datastore concerné via l'interface Web du PBS. Il sert à automatiser le nettoyages des sauvegardes sur le Datastore.

Backup Job​

Le Backup Job se définit lui sur le PVE et permet de séléctionner les machines à sauvegarder automatiquement en séléctionant comme destination le Datastore du PBS.

Résultat​

Bien Joué ! ✔️

Félécitations vous venez de terminer la mise en place d'une solution de sauvegarde pour vos environnement virtualisé PVE.

Pour vérifiez que vos sauvegardes ont bien été effectués rendez vous dans le contenu de votre Datastores
