Veille Technologique & Cyber Sécurité avec FreshRSS

Introduction & Prérequis​

Bienvenue dans ce guide de mise en place de Fresh RSS sur Docker à l'aide de Docker Compose.
Fresh RSS est un agrégateur de flux RSS auto-hébergé qui vous permet de centraliser et de lire vos flux RSS préférés en toute simplicité.
En utilisant Docker, vous pouvez rapidement déployer Fresh RSS sur votre serveur sans avoir à vous soucier de la configuration de l'environnement.
Ce guide vous fournira des instructions pas à pas pour créer et exécuter un conteneur Docker Fresh RSS, en garantissant que vos flux RSS restent accessibles et sécurisés.
Suivez attentivement les étapes suivantes pour configurer votre propre instance de Fresh RSS et profiter de la commodité de la gestion de vos flux RSS.

Prérequis​

Dans ce guide je vais déployer Fresh RSS à l'aide de Docker et d'un fichier docker-compose.yaml. Docker sera instalé sur un conteneur Proxmox qui utilise un template standart de Debian 12.

  • Accès SSH à votre Machine
  • Docker & docker-compose
  • Accès Internet

Installation Docker​

Dans un premier temps on va installer Docker sur notre machine mais d'abord mettons la à jour !

apt update && apt upgrade
apt install docker.io -y && apt install docker-compose -y

On va maintenant créer un dossier appeler docker où seront stockés les fichiers de nos conteneurs.

cd /
mkdir docker
STOCKAGE 📦

Dans ce guide :

  • On stocke le contenu du conteneur en local sur notre machine.

Le TOP ?

  • Sur mon infra de prod 🚀, je stocke les données de mes conteneurs dans un partage NFS sur mon NAS. Dans ces cas-là pas de soucis pour supprimer une machine. Il me suffit de remonter le partage sur la nouvelle machine et de re-déployer mes conteneurs qui retrouveront leurs données.

Fichier YAML​

Désormais on va dans notre dossier docker et on créer un fichier appeler docker-compose.yaml et on le modifie avec ce contenu :

#FRESH RSS
version: '3.3'
services:
    freshrss:
        container_name: freshrss
        environment:
            - PUID=1000
            - PGID=100
            - TZ=Europe/Paris
        ports:
            - '8085:80'
        volumes:
            - '/docker/freshrss/data:/var/www/FreshRSS/data'
            - '/docker/freshrss/extensions:/var/www/FreshRSS/extensions'
        restart: unless-stopped
        image: freshrss/freshrss

On sauvegarde et on va maintenant déployer tout ça !

Déploiement Conteneur​

Pour déployer le contenu que l'on a renseigné dans notre fichier on va utiliser une commande

dokcer-compose up -d

Si vous êtes dans le dossier docker là où vous avez créer votre .yaml alors le déploiement se déroule sans embûche.

Web Interface​

Installation​

Maintenant que votre conteneur est opérationnel rendez vous sur l'adresse IP de votre machine avec un navigateur en spécifiant le port 8085.
Ex : http://192.168.XXX.XXX:8085

Thèmes​

Pour personnaliser l'affichage de votre instance et la rendre plus agréable vous pouvez choisir parmis une selection de thèmes.
Pour ça allez dans les paramètres en haut à droite et rendez vous dans la catégorie Affichage chosissez votre thême et valider en bas de la page.

Extensions​

FreshRSS propose plusieurs extensions pour son projet. Celle que noue allons installer s'apelle YouTubeChannel2RssFeed et c'est une extensions qui permet de transformer des URLs YouTube en flux RSS.
Vous pouvez mettre une chaîne YouTube ou un link de PlayList.
Pour installer cette extension rendez vous dans le dossier extensions de FreshRSS dans mon cas : /docker/freshrss/extensions/ Télécharger le fichier master.zip contenant un lots d'extensions.

wget https://github.com/cn-tools/cntools_FreshRssExtensions/archive/refs/heads/master.zip

Installer l'utilitaire Unzip

apt install unzip

Dézippez le fichier

unzip master.zip
cd cntools_FreshRssExtensions-master/

Choix de notre extension

mv xExtension-YouTubeChannel2RssFeed/ ..
cd ..

Petit nettoyage

rm master.zip
rm -rf cntools_FreshRssExtensions-master/

Et voilà l'extension est désormais installé il n'y plus qu'a l'activer.

Conclusion​

Bien Joué ! ✔️

Voilà vous avez réussi à mettre en place FreshRSS vous allez désormais pouvoir effectuer une veille de qualité !
