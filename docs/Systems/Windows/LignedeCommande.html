Ligne De Commandes (CLI)

Introduction & Prérequis​

Bienvenue dans ce guide détaillé des commandes Batch pour le terminal Windows CMD. Que vous soyez débutant ou utilisateur expérimenté, ces commandes vous aideront à naviguer et à manipuler les fichiers et les dossiers dans un environnement DOS.

CLI / Terminal

Pour exécuter des commandes sur votre Machine Windows ouvrez le terminal CMD dans la barre de Recherche ou utiliser le Raccourci Windows + R

note

Sous Windows une commande se présente sous cette forme : Commande [/-options] [arguments]
Pour avoir l'aide d'une commande on fait un /?

Les Chemins​

  • Chemin Relatif = Un tel chemin décrit la position d'un fichier par rapport au répertoire courant. Ne commence pas par \ ❌
  • Chemin Absolu = Un tel chemin décrit la position d'un fichier par rapport à la racine du lecteur. Commence par \ ✔️
info

DOS n'est pas sensible à la casse (MAJ ou min). C'est à dire que vous pouvez utiliser les lettres majuscules et les lettres minuscules sans soucis 🙂

Le PATH​

Le PATH ou chemin de recherche définit la liste des répertoires où l’interpréteur de commandes doit chercher les fichiers exécutables.

Comment éditer le PATH ?

Tapez dans votre barre de recherche "variables d'environnement" vous arrivez sur une fenêtre propriété systèmes avec un bouton variables d'environnement en bas.

Les caractères Joker​

Un métacaractère (en anglais, wildcard ou joker) est un type de caractère informatique utilisé lors de la recherche d'un mot ou d'une expression incomplète sur un réseau informatisé, ordinateur ou internet.
Le métacaractère remplace généralement la fin ou le début du mot recherché. 🃏

Utilisation du Point d'Interogation : ?

Le point d'interrogation remplace n'importe quel caractère.

Exemples de Caractères Joker
  • Une commande recevant comme paramètre archive.r?? va s'exécuter sur les fichiers archive.raw , archive.r00 , archive.r01 , (mais non pas sur archive.rp).
    L'astérisque remplace n'importe quelle suite de caractères.
  • Une commande recevant comme paramètre *.txt va s'exécuter sur les fichiers config.txt, hello.txt, a01.txt , etc.

Les Attributs de Fichier​

Les fichiers sur Windows possèdent des attributs qui peuvent changer le comportement ou l’affichage depuis l’explorateur de fichiers. 📁
On peut alors afficher l’attribut d’un fichier et bien entendu le modifier.
Ils sont associés au nom des fichiers

Les Fichiers de Commandes​

Il s'agit essentiellement de fichiers texte avec l'extension .bat ou .cmd qui sont exécutés depuis l'interface en ligne de commande comme traitement par lots.
Ces fichiers sont généralement créés pour effectuer des tâches de routine et pour lancer d'autres programmes dans le processus. Exemple d'un fichier .bat qui s'appelle archive.bat

COPY %1 C:\Sauvegarde\ 
ATTRIB -a %1 
ECHO Le fichier %1 a été archivé.
Editeur

Pour créer, éditer votre fichier de commandes BATCH vous avez la possibilité d'utiliser différent éditeur de textes comme celui de base mais aussi Notepadd++ ou même Visual Studio Code.
