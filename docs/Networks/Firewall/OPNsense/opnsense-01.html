Installation de OPNsense sur VM (PVE)

Introduction & Prérequis​

OPNsense est un système d'exploitation open source basé sur FreeBSD, conçu pour servir de pare-feu et de routeur. Il offre une gamme complète de fonctionnalités de sécurité et de gestion du réseau. Ce guide vous expliquera comment installer OPNsense sur votre matériel, que ce soit un ordinateur dédié ou une machine virtuelle.

Il est très similaire sur différents aspects à PfSense car c'est un Fork de PfSense.

Ce guide sera donc lui aussi très semblable à celui de l'installation d'une VM PfSense. Guide PfSense

Prérequis​

Les Prérequis d'une Machine Virtuelle OPNsense sous PVE sont les suivants :

  • Une Machine Virtuelle avec (2GO de RAM, 8GO de Stockage, 1 Core)
  • L'ISO officiel de la dernière version de OPNsense (Actuellement 24.1)
  • 3 Interfaces Réseau dans le cas d'une DMZ (Sur PVE ce sont des vmbr)
  • Une Machine Windows / Linux avec un navigateur pour l'interface Web de OPNsense qui est dans le LAN

Création de la VM​

Démarrons sans plus tarder la création de notre machine virtuelle sous PVE qui hébergera notre Firewall OPNsense. Les Paramètres que je vais utiliser ont été présentés dans les Prérequis. Ce sont des valeurs minimales à adapté en fonction de vos besoins.

  • Partie OS : Séléction de l'ISO pas de changement sur la catégorie Guest OS
  • Partie System : Tout par défaut pas de changement
  • Partie Disks : Taille de 8GB
  • Partie CPU : 1 Core
  • Partie Memory : 2048 MIB
  • Partie Network : Selectionnez le vmbr côté WAN qui vous fournira internet

A la fin de la création de votre machine virtuelle selectionnez la et dans la section Hardware ajoutez une carte réseau pour le LAN en séléctionnant un vmbr libre

Lancement de la VM​

Notre machine est prête à être lancé on va donc sans plus tarder l'éxécuter !
Voici le premier affcichage de OPNsense !

Patientez un court instant avant de pouvoir configurer OPNsense.

Dès que la phrase : Press Any Key to start the manual interface assignement apparait alors appuyer sur ENTER et répondez aux questions de cette manière.

Do you want to configure LAGGs now ?[y/N] : N
Do you want to configure VLANs now ?[y/N] : N

Ensuite assignez vos interfaces WAN/LAN aux vtnet concernés.

WAN | vtnet0
LAN | vtnet1

Validez vos choix.
OPNsense va alors écrire cette configuration.

Installation sur le Disque de OPNsense​

Après avoir écrit la configuration on va donc pouvoir procéder à l'installation de cette configuration sur notre disque.
Pour cela on se connecte à OPNsense avec les identifiants suivants :

LOGIN : installer
PASSWORD : opnsense

En se connectant avec ces identifiants cela déclenche le processus d'installation d'OPNsense

Selection du Clavier on descend un peu et on selectionne French

Section de l'install UFS

Selection du Disk de destination

Ensuite un court chargement s'effectue il permet d'écrire notament les tables de partitions sur notre disque

Puis l'installation démarre !

A la fin de l'installation on selectionne l'option Complete Install car le changement du Password ROOT peut aussi se faire dans l'interface Web.

Interface Web​

Désormais OPNsense est bien installé on va pouvoir se connecter sur l'interface Web pour gérer OPNsense. Dans mon cas l'interface LAN à une adresse en 192.168.1.1/24 donc je veille à avoir une même adresse dans ce résau pour avoir accès à l'interface Web.

En se rendant donc à l'adresse http://192.168.1.1 on arrive sur l'interface de connexion.

Puis on se connecte avec les identifiants suivants :

USERNAME : root
PASSWORD : opnsense

Ensuite un wizard va se lancer au bout de quelques secondes.

  • General Information | C'est ici qu'on va rensigner le nom de notre équipement le domaine au quel le joindre, la langue de l'interface web et les serveurs DNS.

  • Ensuite on gère l'heure de notre équipement

  • L'interface WAN - Dans mon cas aucune modification car je la laisse en DHCP

  • L'interface LAN - Définition de l'adresse IP de l'interface LAN

  • Puis on définit un nouveau mot de passe sécurisé pour le compte Root

Ensuite on relance OPNsense ...

Voià nous avons fini la configuration de OPNsense

Résultat​

Bien Joué ! ✔️

Félicitations ! Vous avez maintenant installé avec succès OPNsense sur votre système. Vous pouvez désormais commencer à configurer votre pare-feu et votre routeur en fonction de vos besoins spécifiques à l'aide de l'interface web conviviale d'OPNsense. Explorez les fonctionnalités avancées d'OPNsense pour renforcer la sécurité et la gestion de votre réseau.
